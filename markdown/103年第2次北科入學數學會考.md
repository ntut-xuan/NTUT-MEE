### 103-02-01

#### Statement

若$f(x)  = 12x^2-26x+5$，則$f(\dfrac{3+\sqrt{2}}{2}) = ?$

$(A)\quad -2+5\sqrt{2}$

$\color{red}(B)\quad -1+5\sqrt{2}$

$(C)\quad 1+5\sqrt{2}$

$(D)\quad 9$

$(E)\quad 2+5\sqrt{2}$



#### Solution

利用綜合除法，將式子轉成以$(3x-2)$表示的形式。

![image-20210712160848023](https://i.imgur.com/2Q0SNFt.png)

綠色字：將$(x-\dfrac{3}{2})$轉換成$(2x-3)$，因此係數的部分要除以$2$。

得到$f(x) = 3(2x-3)^2+5(2x-3)-7$，因此$f(\dfrac{3+\sqrt{2}}{2}) = 6 + 5\sqrt{2}-7 = 5\sqrt{2}-1$，故選$(B)$



### 103-02-02

#### Statement

若$f(\dfrac{2+x}{4-x}) = x+2$，則$f(a) = ?$

$(A)\quad \dfrac{6a}{1-a}$

$(B)\quad \dfrac{2+a}{2-a}$

$(C)\quad \dfrac{4+a}{2-a}$

$(D)\quad \dfrac{2-a}{2+a}$

$\color{red}(E)\quad \dfrac{6a}{1+a}$



#### Solution

$\dfrac{2+x}{4-x} = a$，得到$x = \dfrac{4a-2}{a+1}$

因此$f(a) = \dfrac{4a-2}{a+1}+2 = \dfrac{6a}{a+1}$，故選$(E)$



### 103-02-03

#### Statement

設$\Delta ABC$中，$\overline{AB} = 6$，$\overline{BC} = 5$，$\overline{CA} = 4$，若$D$為$\overline{BC}$上一點使$\overline{AD} = 4$，則$\overline{BD} = ?$

$(A)\quad 1$

$(B)\quad 2$

$(C)\quad \dfrac{5}{2}$

$(D)\quad 3$

$\color{red}(E)\quad 4$



#### Solution

$\cos C = \dfrac{5^2+4^2-6^2}{2\times 5\times 4} = \dfrac{1}{8}$

可以推得$\cos C = \cos \angle ACD = \dfrac{4^2+(\overline{CD})^2 - 4^2}{2\times4\times \overline{CD}} = \dfrac{1}{8}$，可得$\overline{CD} = 1$

因此$\overline{BD} = \overline{BC} - \overline{CD} = 5 - 1 = 4$，故選$(E)$



### 103-02-04

#### Statement

求圓$x^2+y^2-4x+6y-3=0$與直線$3x-4y=-7$的最近距離為何？

$(A)\quad 0$

$\color{red}(B)\quad 1$

$(C)\quad 2$

$(D)\quad 4$

$(E)\quad 5$



#### Solution

$x^2+y^2-4x+6y-3=0$

利用配方法，得到$(x-2)^2+(y+3)^2=16$

因此得到圓心$(2, -3)$，且半徑$r = \sqrt{16} = 4$



考慮圓心與直線的最近距離，利用點到直線距離公式

$\dfrac{|3(2)-4(-3)+7|}{\sqrt{3^2+(-4)^2}} = \dfrac{|25|}{5} = 5$ 

長度大於半徑，因此與圓的最近距離為$5 - 4 = 1$，故選$(B)$

### 103-02-08

#### Statement

設雙曲線之方程式為$\dfrac{x^2}{4}-y^2=1$，若將此雙曲線之貫軸長放大為$2$倍，共軛軸與中心點不變，則此雙曲線方程式變為何？

$(A)\quad \dfrac{x^2}{8}-y^2=1$

$(B)\quad \dfrac{x^2}{8}-\dfrac{y^2}{2}=1$

$\color{red}(C)\quad \dfrac{x^2}{16}-y^2=1$

$(D)\quad \dfrac{x^2}{16}-\dfrac{y^2}{2} = 1$

$(E)\quad \dfrac{x^2}{8}-\dfrac{y^2}{4} = 1$



#### Solution

又共軛軸與中心點不變，則此方程式變為$\dfrac{x^2}{16}-y^2=1$，故選$(C)$



### 103-02-09

#### Statement

不等式$\dfrac{1+2\log_2 x}{-1+\log_2 x} \le 1$之解為何？

$(A)\quad \dfrac{1}{2}\le x\le 2$

$(B)\quad \dfrac{1}{2}\le x < 2$

$(C)\quad \dfrac{1}{4}\le x \le 2$

$\color{red}(D)\quad \dfrac{1}{4}\le x < 2$

$(E)\quad 1 \le x < 2$



#### Solution

$\dfrac{1+2\log_2 x}{-1+\log_2 x} \le 1$

$\Rightarrow \dfrac{1+2\log_2 x}{-1+\log_2 x} - 1 \le 0$

$\Rightarrow \dfrac{2+\log_2 x}{-1+\log_2 x} \le 0$

因此我們考慮以下兩種情況取聯集

$\left\{\begin{array}\ 2+\log_2 x \le 0 \\ -1 + \log_2 x > 0\end{array}\right. \Rightarrow x \in \varnothing$

$\left\{\begin{array}\ 2+\log_2 x \le 0 \\ -1 + \log_2 x < 0 \end{array}\right. \Rightarrow [\dfrac{1}{4}, 2)$

兩種情況取聯集，得到$\dfrac{1}{4} \le x < 2$，故選$(D)$



### 103-02-10

#### Statement

設$\sin\theta - \cos\theta = \dfrac{1}{2}$，求$\cos4\theta$

$\rm (A)\ \dfrac{-1}{4}$

$\color{red}{\rm(B)\ \dfrac{-1}{8}}$

$\rm(C)\ \dfrac{\sqrt{3}}{4}$

$\rm(D)\ \dfrac{\sqrt{3}}{2}$

$\rm (E)\ 1$



#### Solution

展開$\cos4\theta$的式子

  $\cos4\theta = \cos^22\theta - \sin^22\theta$

  $= \cos^22\theta - (2\sin\theta\cos\theta)^2$

將$\sin\theta-\cos\theta$平方，得到

  $(\sin\theta-\cos\theta)^2 = \dfrac{1}{4}$

  $\Rightarrow \sin^2\theta - 2\sin\theta\cos\theta + \cos^2\theta = \dfrac{1}{4}$

  $\Rightarrow 1 - 2\sin\theta\cos\theta = \dfrac{1}{4}$

  $\Rightarrow \sin\theta\cos\theta = \dfrac{3}{8}$

求$\sin^22\theta$

  $\sin^22\theta = (\dfrac{6}{8})^2 = (\dfrac{3}{4})^2 = \dfrac{9}{16}$

求$\cos^22\theta$

  $\sin^22\theta + \cos^22\theta = 1,\ \cos^22\theta = 1 - \dfrac{9}{16} = \dfrac{7}{16}$

計算$\cos^22\theta - \sin^22\theta$

  $\cos^22\theta - \sin^22\theta = \dfrac{7}{16} - \dfrac{9}{16} = \dfrac{-2}{16} = \dfrac{-1}{8}$

因此$\cos4\theta = \dfrac{-1}{8}$，故選$(B)$



### 103-02-11

#### Statement

若直線通過點$P(3, 4)$且與兩座標軸在第一象限圍成三角形，則此三角形面積最小值為何？

$(A)\quad 10$

$(B)\quad 12$

$(C)\quad 14$

$(D)\quad 20$

$\color{red}(E)\quad 24$



#### Solution

設直線截距式$\dfrac{x}{a}+\dfrac{y}{b}=1$，三角形面積為$\dfrac{ab}{2}$

因為與第一象限圍成區域面積，所以$a, b>0$

代入點$P(3, 4)$得到$\dfrac{3}{a}+\dfrac{4}{b} = 1$

利用算幾不等式，$\dfrac{\dfrac{3}{a}+\dfrac{4}{b}}{2} \ge \sqrt{\dfrac{12}{ab}}$，解得$ab$最小值為$48$

因此$\dfrac{48}{2} = 24$，故選$(E)$



### 103-02-12

#### Statement

已知兩圓$x^2+y^2=10$與$x^2+y^2-2x+4y=5$有兩交點，則此兩交點的距離為何？

$(A)\quad 3$

$(B)\quad 4$

$(C)\quad \sqrt{26}$

$\color{red}(D)\quad \sqrt{35}$

$(E)\quad 5\sqrt{2}$



#### Solution

利用配方法，可知$x^2+y^2-2x+4y=5 \Rightarrow (x-1)^2+(y+2)^2=10$

設圓心$O_1 = (0, 0)$，$O_2 = (1, -2)$，且兩圓交於$A, B$，其中$\overline{O_1A}=\overline{O_1B}=\overline{O_2A}=\overline{O_2B} = \sqrt{10}$

已知$\angle AO_1 B + \angle O_1AO_2 = 180^{\circ}$

因此$\cos(\angle O_1AO_2) = \cos(\pi-\angle AO_1B) = -\cos(\angle AO_1B)$

因此$\dfrac{(\sqrt{10})^2+(\sqrt{10})^2-(\overline{AB})^2}{2\times\sqrt{10}\times \sqrt{10}} = -\dfrac{(\sqrt{10})^2+(\sqrt{10})^2-(\sqrt{5})^2}{2\times\sqrt{10}\times\sqrt{10}}$

可知$\overline{AB} = \pm\sqrt{35}$ (負不合)，故選$(D)$



### 103-02-13

#### Statement

$1\times 2+2\times 3+3\times 4+...+25\times 26 =?$

$(A)\quad 5125$

$\color{red}(B)\quad 5850$

$(C)\quad 6500$

$(D)\quad 6975$

$(E)\quad 7200$



#### Solution

$1\times 2+2\times 3+3\times 4+...+25\times 26$

$= \displaystyle \sum^{25}_{x=1} x(x+1) = \sum^{25}_{x=1} (x^2+x) = \sum^{25}_{x=1} x^2 + \sum^{25}_{x=1}x$

$= \dfrac{25(25+1)(50+1)}{6} + \dfrac{25\times 26}{2}$

$= 25\times 13\times 17 + 25\times 13$

$= 5525+325 = 5850$，故選$(B)$

設曲線$y=\log_3 x$與$x$軸、直線$x=9$的交點分別為$A$、$B$，且直線$x = 9$與$x$軸的交點為$C$，則$\Delta ABC$的面積為何？

$\color{red}(A)\quad 8$

$(B)\quad 12$

$(C)\quad 16$

### 103-02-18

#### Statement

設$\vec{a}=<\cos\alpha, \sin\alpha>$，$\vec{b} = <\cos\beta, \sin\beta>$，且$0 < \alpha < \pi$，$0 < \beta < \pi$，$\alpha \neq \beta$，則兩向量$\vec{a}+\vec{b}$與$\vec{a}-\vec{b}$的夾角為何？

$(A)\quad 0$

$(B)\quad \dfrac{\pi}{4}$

$(C)\quad \dfrac{\pi}{3}$

$\color{red}(D)\quad \dfrac{\pi}{2}$

$(E)\quad \pi$



#### Solution

由題目可知，兩向量是以原點為起點，單位圓上為終點做兩向量。

若任意取圓上兩點$B, C$，則向量$\vec{a}$為原點至點$B$的向量，且向量$\vec{b}$為原點至點$C$的向量。

因此根據向量平行四邊形，可知$\vec{a}+\vec{b}$為$\angle CAB$的角平分向量，且$\vec{a}-\vec{b}$為$\vec{CB}$

由三角形的角平分線性質可知$\overline{CB}$會被分成兩段

再由圓性質，過圓心的一射線能將圓上任意兩點等分，則射線跟線段一定為直角，故選$(D)$

### 103-02-20

#### Statement

若$\displaystyle \lim_{x\rightarrow 0}\dfrac{\sqrt{ax+b}-2}{x} = \dfrac{-1}{4}$，求$a+b=?$

$\color{red}(A)\quad 3$

$(B)\quad 4$

$(C)\quad 5$

$(D)\quad 6$

$(E)\quad 7$



#### Solution

$\displaystyle \lim_{x\rightarrow 0}\dfrac{\sqrt{ax+b}-2}{x} = \lim_{x\rightarrow 0}\dfrac{ax+b-4}{x(\sqrt{ax+b}+2)}$

若極限存在，則$b-4=0$，因此$b = 4$

$\displaystyle \lim_{x\rightarrow 0}\dfrac{ax}{x(\sqrt{ax+4}+2)} = \lim_{x\rightarrow 0}\dfrac{a}{(\sqrt{ax+4}+2)} = \dfrac{-1}{4}$

得到$\dfrac{a}{4} = \dfrac{-1}{4}$，因此$a= -1$

因此$a+b=4-1=3$，故選$(A)$
