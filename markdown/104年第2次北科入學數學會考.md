### 104-02-01

#### Statement

已知$\Delta ABC$中，$\angle BCA = 120^{\circ}$，$\overline{AC} = 3$，$\overline{BC}=5$且$D$在$\overline{AB}$上。

若$\overline{CD} \perp \overline{AB}$，則$\overline{CD}=?$

$(A)\quad \dfrac{5\sqrt{3}}{14}$

$\color{red}(B)\quad \dfrac{15\sqrt{3}}{14}$

$(C)\quad \dfrac{35\sqrt{3}}{2}$

$(D)\quad \dfrac{55\sqrt{3}}{2}$

$(E)\quad \dfrac{75\sqrt{3}}{2}$



#### Solution

利用餘弦定理求$\overline{AB}$

$\cos120^{\circ} = \dfrac{5^2+3^2-(\overline{AB})^2}{2\times 5\times 3} = -\dfrac{1}{2}$，可以算出$\overline{AB} = \pm 7$ (負不合)。

利用正弦來算出面積

$\Delta ABC = \dfrac{1}{2}\times 3\times 5\times \sin 120^{\circ} = \dfrac{15\sqrt{3}}{4} = \dfrac{1}{2}\times 7\times \overline{CD}$

因此可以求出$\overline{CD} = \dfrac{15\sqrt{3}}{14}$，故選$(B)$



### 104-02-02

#### Statement

設$\dfrac{7x^2-13x}{x^3-x^2-x+1} = \dfrac{A}{x-1} + \dfrac{B}{(x-1)^2} + \dfrac{C}{x+1}$，則$A+B+C=?$

$(A)\quad 2$

$\color{red}(B)\quad 4$

$(C)\quad 6$

$(D)\quad 8$

$(E)\quad 10$



#### Solution

$\dfrac{7x^2-13x}{x^3-x^2-x+1} = \dfrac{A}{x-1} + \dfrac{B}{(x-1)^2} + \dfrac{C}{x+1}$

$\Rightarrow \dfrac{7x^2-13x}{(x-1)^2(x+1)} = \dfrac{A}{x-1} + \dfrac{B}{(x-1)^2} + \dfrac{C}{x+1}$

$\Rightarrow 7x^2-13x = A(x-1)(x+1) + B(x+1) + C(x-1)^2$

代入$x = 1$，得$2B = -6$，因此$B=-3$

$7x^2-10x+3 = A(x-1)(x+1) + C(x-1)^2$

代入$B=-1$，得$4C = 20$，因此$C=5$

$2x^2-2 = A(x-1)(x+1) = A(x^2-1)$，可得$A=2$

因此$A+B+C=2-3+5=4$，故選$(B)$



### 104-02-03

#### Statement

若扇形的夾角為$\theta$，弧長為$\dfrac{\pi}{2}$，面積為$\dfrac{3}{2}\pi$，則$\theta = ?$

$\color{red}(A)\quad \dfrac{\pi}{12}$

$(B)\quad \dfrac{\pi}{10}$

$(C)\quad \dfrac{\pi}{8}$

$(D)\quad \dfrac{\pi}{6}$

$(E)\quad \dfrac{\pi}{3}$



#### Solution

由弧長為$\dfrac{\pi}{2}$可知，$r\theta = \dfrac{\pi}{2}$

由面積為$\dfrac{3\pi}{2}$可知，$\dfrac{1}{2}r^2\theta = \dfrac{3}{2}\pi$

兩式$\left\{\begin{array}\ r\theta = \dfrac{\pi}{2} \\ \dfrac{1}{2}r^2\theta = \dfrac{3}{2}\pi\end{array}\right.$可知，$r = 6$且$\theta = \dfrac{\pi}{12}$，故選$(A)$

$\color{red}(B)\quad 11$

### 104-02-07

$(C)\quad 2$

$(D)\quad 3$

$\color{red}(E)\quad 6$

### 104-02-10

#### Statement

設$\sin x -2\cos x = \sqrt{5}$，則$\sin x + 2\cos x = ?$

$(A)\quad -1$

$\color{red}(B)\quad \dfrac{-3}{\sqrt{5}}$

$(C)\quad 0$

$(D)\quad \dfrac{3}{\sqrt{5}}$

$(E)\quad 1$



#### Solution

$\sin x - \sqrt{5} = 2\cos x$

$\Rightarrow \sin x - \sqrt{5} = 2\sqrt{1-\sin^2 x}$

$\Rightarrow \sin^2 x - 2\sqrt{5}\sin x + 5 = 4(1-\sin^2 x)$

$\Rightarrow 5\sin^2 x -2\sqrt{5}\sin x + 1 = 0$

$\Rightarrow (\sqrt{5}\sin x - 1)^2 = 0$

$\Rightarrow \sin x = \pm\dfrac{1}{\sqrt{5}}$(負不合)

$\Rightarrow \cos x = \pm\dfrac{2}{\sqrt{5}}$(正不合)

因此$\sin x + 2\cos x = \dfrac{-3}{\sqrt{5}}$，故選$(B)$

### 104-02-14

#### Statement

不等式$2^{2x+1} - 7\cdot 2^x - 4 \ge 0$的解集合為何？

$(A)\quad (-\infty, -1]$

$(B)\quad (-1, 2]$

$(C)\quad [-1, 2]$

$(D)\quad [-1, 2)$

$\color{red}(E)\quad [2, \infty)$



#### Solution

$2^{2x+1} - 7\cdot 2^x - 4 \ge 0$

$\Rightarrow 2\cdot 2^{2x} -7\cdot 2^x - 4 \ge 0$

令$t = 2^x$，則$2t^2-7t-4\ge 0$

因此$(2t+1)(t-4)\ge0$，得到$(-\infty, -\dfrac{1}{2}] \cup [4, \infty)$

由於$2^x$的域值為正整數，因此$(-\infty, -\dfrac{1}{2}]$不合，考慮$[4, \infty)$

可知$t \ge 4$，還原後得到$x \ge 2$，故選$(E)$



### 104-02-15

#### Statement

若$f(x) = \dfrac{1}{\sqrt{4-x}}$，$g(x) = \sqrt{x-3}$，則$f$與$g$的合成函數$f\circ g$的定義域為何？

$(A)\quad (3, 4)$

$(B)\quad [3, 4]$

$(C)\quad (3, 19)$

$\color{red}(D)\quad [3, 19)$

$(E)\quad (3, 19]$



#### Solution

$f(g(x)) = \dfrac{1}{\sqrt{4- \sqrt{x-3}}}$

考慮$\sqrt{4-\sqrt{x-3}} \neq 0$，則$x \neq 19$

考慮$4-\sqrt{x-3} > 0$，則$x < 19$

考慮$\sqrt{x-3} \ge 0$，則$x \ge 3$

三者取交集，得到$[3, 19)$，故選$(D)$



### 104-02-16

#### Statement

在$\Delta ABC$中，向量$\vec{AB} = <1, 2>$，$\vec{AC} = <x-1, x>$，$x > 0$，若$\Delta ABC$之面積為$\dfrac{5}{2}$，則$x=?$

$(A)\quad 1$

$(B)\quad 2$

$(C)\quad 3$

$(D)\quad 5$

$\color{red}(E)\quad 7$



#### Solution 1

> 速解 by Trava

$\dfrac{1}{2}\left|\begin{array}\ & 1 & 2 & \\ &x-1 & x & \end{array} \right| = \dfrac{5}{2}$

因此$5 = |x-2x+2|$

$\Rightarrow 5 = |-x+2|$

由於$x>0$，所以$-5 = -x+2$，得到$x = 7$，故選$(E)$



#### Solution 2

> 很仔細的解 by Uriah

考慮到以$\vec{AB}$與$\vec{AC}$所建立的平行四邊形，其面積為$|\vec{AB}||\vec{AC}|\sin \theta$

又$\cos\theta = \dfrac{\vec{AB}\cdot\vec{AC}}{|\vec{AB}||\vec{AC}|}$，因此$\sin\theta = \sqrt{1-\cos^2\theta}$

可得三角形的面積為$\dfrac{1}{2}\sqrt{|\vec{AB}|^2|\vec{AC}|^2 - (\vec{AB}\cdot \vec{AC})^2} = \dfrac{5}{2}$

也就是$\sqrt{5(2x^2-2x+1) - (9x^2-6x+1)} = 5$

$\Rightarrow x^2-4x+4 = 25$，得到$x = -3$(不合)或$x= 7$，故選$(E)$

### 104-02-18

#### Statement

若$\displaystyle \lim_{x\rightarrow 0} \dfrac{\sqrt{ax+b}-3}{x} = \dfrac{-1}{6}$，則$ab=?$

$\color{red}(A)\quad -9$

$(B)\quad -3$

$(C)\quad 1$

$(D)\quad 3$

$(E)\quad 9$



#### Solution

$\displaystyle \lim_{x\rightarrow 0} \dfrac{\sqrt{ax+b}-3}{x} = \lim_{x\rightarrow 0}\dfrac{ax+b-9}{x(\sqrt{ax+b}+3)}$

如果極限存在，則$b-9$必須要等於0，因此$b=9$

故$\displaystyle \lim_{x\rightarrow 0}\dfrac{a}{(\sqrt{ax+9}+3)} = \dfrac{-1}{6}$

得到$a = -1$，此$ab = (-1)\cdot 9 = -9$，故選$(A)$



### 104-02-19

#### Statement

設$f(x) = \dfrac{x}{2x-1}$，則$\displaystyle \lim_{h\rightarrow 0} \dfrac{f(1+2h)-f(1)}{h} = ?$

$\color{red}(A)\quad -2$

$(B)\quad -1$

$(C)\quad \dfrac{-1}{2}$

$(D)\quad \dfrac{1}{2}$

$(E)\quad 1$



#### Solution

$\displaystyle \lim_{h\rightarrow 0} \dfrac{f(1+2h)-f(1)}{h} = \displaystyle 2\lim_{h\rightarrow 0} \dfrac{f(1+2h)-f(1)}{2h} = 2f'(1)$

$f'(x) = -\dfrac{1}{(2x-1)^2}$，則$2f'(1) = -2$，故選$(A)$



### 104-02-20

#### Statement

將平面曲線$y=f(x)$向右平移1單位，再以$y$軸為中心放大為兩倍，則所得曲線的方程式為何？

$(A)\quad y=f(\dfrac{x-1}{2})$

$\color{red}(B)\quad y = f(\dfrac{x}{2}-1)$

$(C)\quad y = f(2x-1)$

$(D)\quad y = f(\dfrac{x+1}{2})$

$(E)\quad y=f(2x+1)$



#### Solution

向右平移一單位，因此$x$改寫為$(x-1)$

放大為兩倍，因此$(x-1)$改寫為$\dfrac{x}{2}-1$

因此曲線方程式為$f(\dfrac{x}{2}-1)$，故選$(B)$
