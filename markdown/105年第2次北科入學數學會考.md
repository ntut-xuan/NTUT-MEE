## 105年第2次北科入學數學會考

### 105-02-01

#### Statement

若$\dfrac{1}{\alpha}$和$\dfrac{1}{\beta}$為方程式$x + \dfrac{3}{x} = 1$的兩根，則$\alpha^2\beta+\alpha\beta^2=?$

$\color{red}(A)\quad \dfrac{1}{9}$

$(B)\quad \dfrac{1}{3}$

$(C)\quad 1$

$(D)\quad 2$

$(E)\quad 3$



#### Solution

 $x + \dfrac{3}{x} = 1$

$\Rightarrow x^2+3=x$

$\Rightarrow x^2-x+3=0$

利用偉達定理

$\dfrac{1}{\alpha} + \dfrac{1}{\beta} = -(\dfrac{-1}{1}) = 1 = \dfrac{\alpha+\beta}{\alpha\beta}$

$\dfrac{1}{\alpha\beta} = 3 \Rightarrow \alpha\beta = \dfrac{1}{3}$

$\because \alpha\beta = \dfrac{1}{3}$

$\therefore \alpha+\beta = \dfrac{1}{3}$

$\alpha^2\beta+\alpha\beta^2 = \alpha\beta(\alpha+\beta) = \dfrac{1}{3}\times \dfrac{1}{3} = \dfrac{1}{9}$，故選$(A)$



### 105-02-02

#### Statement

下列何者為方程式$4(4^x+4^{-x}) - (2^x+2^{-x}) - 60 = 0$的解？

$(A)\quad -4$

$\color{red}(B)\quad -2$

$(C)\quad 0$

$(D)\quad 1$

$(E)\quad 3$



#### Solution

$4(4^x+4^{-x})-(2^x+2^{-x})-60 = 0$

$\Rightarrow 4(2^{2x}+2^{-2x})-(2^x+2^{-x})-60 = 0$

令$t = (2^x+2^{-x})$，則$t^2 = 2^{2x}+2^{-2x}+2$

將式子轉成$4t^2-t-68=0$

得到$t = -4$(不合)與$t =\dfrac{17}{4}$

還原$t$，得到$x = \pm 2$，故選$(B)$



### 105-02-03

#### Statment

若$0 \le \theta \le 2\pi$，則滿足方程式$\dfrac{\cos\theta(1+2\sin\theta)}{\cos^2\theta-\sin^2\theta-\sin\theta} = 0$之$\theta$的個數為何？

$(A)\quad 0$

$(B)\quad 1$

$(C)\quad 2$

$\color{red}(D)\quad 3$

$(E)\quad 4$



#### Solution

$\cos\theta(1+2\sin\theta) = 0$

$\cos\theta = 0 \Rightarrow \theta = 90^{\circ}, 270^{\circ}$

$(1+2\sin\theta) = 0 \Rightarrow \sin\theta = \dfrac{-1}{2} \Rightarrow \theta = 240^{\circ}, \theta = 300^{\circ}$

考慮$\cos^2\theta - \sin^2\theta - \sin\theta$的定義域，要求$\cos^2\theta - \sin^2\theta - \sin\theta \neq 0$

因此$-2\sin^2\theta - \sin\theta + 1 \neq 0$

得到$\sin\theta \neq -1, \dfrac{1}{2}$

因此$\sin\theta \neq 270^{\circ}, 60^{\circ}, 120^{\circ}$

故答案有$3$個，故選$(D)$



### 105-02-04

#### Statement

已知$f(x)$為二次多項式。若$f(x) < 0$之解為$-3 < x < 2$且$f(1) = -4$，則$f(x) = ?$

$(A)\quad x^2-3x+2$

$(B)\quad x^2-2x-3$

$\color{red}(C)\quad x^2+x-6$

$(D)\quad x^2+2x-7$

$(E)\quad x^2+3x-8$



#### Solution

因為$f(x) < 0$之解為$-3 < x < 2$，因此$f(x)$可以列式成$f(x) = a(x+3)(x-2)$

又$f(1) = -4$，因此$a\times 4\times -1  = -4$，得到$a =1$

因此$f(x) = (x+3)(x-2) = x^2+x-6$，故選$(C)$



### 105-02-05

#### Statement

已知一正方體所有邊長和為$x$。若將此正方體的表面積和體積之值加總表示成一多項式，則此多項式包含哪一種因式？

$(A)\quad x+6$

$(B)\quad x+12$

$(C)\quad x+18$

$(D)\quad x+36$

$\color{red}(E)\quad x+72$



#### Solution

所有邊長和為$x$，因此一邊長為$\dfrac{x}{12}$

可知體積為$\dfrac{x^3}{12^3}$且表面積為$\dfrac{x^2}{24}$

因此兩個加總得到$\dfrac{x^3}{1728} + \dfrac{x^2}{24} = \dfrac{x^3+72x^2}{1728} = \dfrac{x^2(x+72)}{1728}$，故有因式$x+72$



### 105-02-06

#### Statement

已知向量$\vec{a} = <5, -12>$，向量$\vec{b}$與向量$\vec{a}$同向且$|\vec{b}|=39$，若$\vec{b}$之起點為$(-1, 3)$，則$\vec{b}$之終點為何？

$\color{red}(A)\quad (14, -33)$

$(B)\quad (-14, 33)$

$(C)\quad (-18, 45)$

$(D)\quad (18, -45)$

$(E)\quad (4, -9)$



#### Solution

向量$\vec{b}$與向量$\vec{a}$同向，因此$\vec{b}=<5a, -12a>, a\in \R$

又$|\vec{b}| = 39$，因此$\sqrt{(5a)^2 + (-12a)^2} = 39$，得到$a = 3$

因此$\vec{b} = <15, -36>$，又$\vec{b}$之起點為$(-1, 3)$

因此$\vec{b}$之終點為$(-1+15, 3-36) = (14, -33)$，故選$(A)$



### 105-02-07

#### Statement

方程式$\log x + \log(x+2) = \log(x+1) + 1$的解為何？

$(A)\quad 4-\sqrt{26}$

$(B)\quad 4-2\sqrt{3}$

$(C)\quad 4-\sqrt{2}$

$(D)\quad 4+2\sqrt{3}$

$\color{red}(E)\quad 4+\sqrt{26}$



#### Solution

$\log x + \log(x+2) = \log(x+1) + 1$

$\Rightarrow \log(\dfrac{x^2+2x}{10x+10}) = 0$

$\Rightarrow \dfrac{x^2+2x}{10x+10} = 1$

$\Rightarrow x^2-8x-10=0$

$\Rightarrow x = \dfrac{8\pm\sqrt{64-4\times1\times -10}}{2} = \dfrac{8\pm2\sqrt{26}}{2} = 4\pm\sqrt{26}$

由於$4 < \sqrt{26}$，因此$4-\sqrt{26} < 0$，不在$\log x$的值域內，因此不合

故$x = 4+\sqrt{26}$，故選$(E)$



### 105-02-08

#### Statement

設$\sin2\theta = \dfrac{4}{5}$，則$\sin^4\theta+\cos^4\theta=?$

$\color{red}(A)\quad \dfrac{17}{25}$

$(B)\quad \dfrac{18}{25}$

$(C)\quad \dfrac{19}{25}$

$(D)\quad \dfrac{4}{5}$

$(E)\quad \dfrac{21}{25}$



#### Solution

$(\sin^2\theta + \cos^2\theta)^2 = \sin^4\theta + 2\sin^2\theta\cos^2\theta + \cos^4\theta$

$=\sin^4\theta + 2(\sin\theta\cos\theta)^2 + \cos^4\theta$

$=\sin^4\theta + \dfrac{1}{2}(\sin^22\theta) + \cos^4\theta = 1$

因此$\sin^4\theta +\cos^4\theta = 1-\dfrac{1}{2}(\dfrac{4}{5})^2 = \dfrac{17}{25}$，故選$(A)$



### 105-02-09

#### Statement

已知$\theta$為平面上兩向量$\vec{a}$與$\vec{b}$之夾角，若$|\vec{a}|=3$、$|\vec{b}|=5$且$|2\vec{a}+\vec{b}|=7$，則$cos\theta=?$

$\color{red}(A)\quad \dfrac{-1}{5}$

$(B)\quad \dfrac{-1}{3}$

$(C)\quad \dfrac{1}{5}$

$(D)\quad \dfrac{1}{3}$

$(E)\quad \dfrac{4}{5}$



#### Solution

$|2\vec{a}+\vec{b}|=7$

$\Rightarrow 4|\vec{a}|^2+4(\vec{a}\cdot\vec{b})+|\vec{b}|^2=49$

$\Rightarrow (\vec{a}\cdot\vec{b}) = -3$

又$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta = -3$

因此$\cos\theta = \dfrac{-3}{15} = \dfrac{-1}{5}$，故選$(A)$



### 105-02-10

#### Statement

若$\dfrac{x^3-5}{x^2-1} = f(x)+\dfrac{a}{x-1}+\dfrac{b}{x+1}$，其中$f(x)$為一次式且$a, b$為常數，求$a+b=?$

$(A)\quad -1$

$(B)\quad 0$

$\color{red}(C)\quad 1$

$(D)\quad 2$

$(E)\quad 3$



#### Solution

假設$f(x) = cx+d$

$x^3-5 = (cx+d)(x^2-1) + a(x+1)+b(x-1)$

$= cx^3-cx+dx^2+ax+a+bx-b$

$= cx^3 + dx^2 + (a-c+b)x + (a-b) = x^3-5$

因此$c = 1, d = 0$

$\left\{\begin{array}\ a+b-1=0\\ a-b = -5\end{array}\right.$，解聯立得$a =-2$且$b = 3$

因此$a+b=1$，故選$(C)$



### 105-02-11

#### Statement

若$\displaystyle \lim_{x\rightarrow 1} \dfrac{\dfrac{1}{a+x}-\dfrac{1}{b}}{x-1} = \dfrac{1}{4}$，則$a+b= ?$

$(A)\quad 0$

$(B)\quad 1$

$(C)\quad 2$

$\color{red}(D)\quad 3$

$(E)\quad 4$



#### Solution

$\displaystyle \lim_{x\rightarrow 1} \dfrac{\dfrac{1}{a+x}-\dfrac{1}{b}}{x-1} = \lim_{x\rightarrow 1}\dfrac{-x+(b-a)}{(ab+bx)(x-1)} = \dfrac{1}{4}$

若極限值存在，則$b-a = 1$，且$ab+b=4$

因此，$\left\{\begin{array}\ b-a=1 \\ ab+b=4 \end{array}\right.$，故得到$(a, b) = (-3, -2)$或$(a, b) = (1, 2)$

$a+b=-5$或$3$，故選$(D)$



### 105-02-12

#### Statement

$\displaystyle \lim_{h\rightarrow 0} \dfrac{(4+3h)^2-4^2}{\sqrt{4+3h}-2} = ?$

$(A)\quad 0$

$(B)\quad 16$

$\color{red}(C)\quad 32$

$(D)\quad 64$

$(E)\quad 不存在$



#### Solution

$\displaystyle \lim_{h\rightarrow 0} \dfrac{(4+3h)^2-4^2}{\sqrt{4+3h}-2} = \lim_{h\rightarrow 0} \dfrac{(4+3h-4)(4+3h+4)}{\sqrt{4+3h}-2} = \lim_{h\rightarrow 0} \dfrac{(4+3h-4)(4+3h+4)(\sqrt{4+3h}+2)}{4+3h-4}$

$\displaystyle \lim_{h\rightarrow 0} (3h+8)(\sqrt{4+3h}+2) = 8\times 4 = 32$，故選$(C)$



### 105-02-13

#### Statement

若拋物線$y=x^2-ax+3$與直線$3x+y+1 = 0$不相交且$a$為整數，則$a$有幾種可能？

$(A)\quad 3$

$(B)\quad 4$

$(C)\quad 5$

$(D)\quad 6$

$\color{red}(E)\quad 7$



#### Solution

已知拋物線的動點為$(x, x^2-ax+3)$

因此考慮代入$3x+y+1$時會大於$0$

得到$3x+x^2-ax+4 > 0 \Rightarrow x^2+(3-a)x+4 > 0$

利用判別式：$(3-a)^2-16 > 0$確認解與$x$無交集

得到$-1 < a < 7$，因此$a$有$7$種可能，故選$(E)$

### 105-02-15

#### Statement

設$f(x) = \dfrac{1}{\sqrt{x-2}}$，$g(x) = 2+x-x^2$，則合成函數$f\circ g$的定義域為何？

$(A)\quad \varnothing (空集合)$

$\color{red}(B)\quad 0 < x < 1$

$(C)\quad 0 \le x \le 1$

$(D)\quad x < 0$或$x > 1$

$(E)\quad x \le 0 或x \ge 1$



#### Solution

$f(g(x)) = \dfrac{1}{\sqrt{2+x-x^2-2}} = \dfrac{1}{\sqrt{-x^2+x}}$

考慮$\sqrt{-x^2+x} \neq 0$，因此$-x^2+x > 0$，因此$0 < x < 1$，故選$(B)$



### 105-02-16

#### Statement

若直線$L$的斜率為$\dfrac{2}{3}$且與$x$軸所夾之銳角為$\theta$，則$\cos(\dfrac{3\pi}{2}-\theta) = ?$

$(A)\quad \dfrac{-3}{\sqrt{13}}$

$\color{red}(B)\quad \dfrac{-2}{\sqrt{13}}$

$(C)\quad \dfrac{-1}{\sqrt{13}}$

$(D)\quad \dfrac{2}{\sqrt{13}}$

$(E)\quad \dfrac{3}{\sqrt{13}}$



#### Solution

直線$L$的斜率為$\dfrac{2}{3}$，因此可知$m = \dfrac{2}{3}$，也就是$\tan\theta = \dfrac{2}{3}$

透過$\tan\theta = \dfrac{2}{3}$，可知$\sin\theta = \dfrac{2}{\sqrt{13}}$且$\cos\theta= \dfrac{3}{\sqrt{13}}$

因此$\cos(\dfrac{3\pi}{2} - \theta) = -\sin\theta = \dfrac{-2}{\sqrt{13}}$，故選$(B)$

### 105-02-18

#### Statement

若一拋物線通過$(-2, 19), (6, 16), (1, 4)$三點，則此拋物線的頂點坐標為何？

$(A)\quad (2, 1)$

$\color{red}(B)\quad (2, 3)$

$(C)\quad (2 ,5)$

$(D)\quad (2, 6)$

$(E)\quad (2, 7)$



#### Solution

由$(-2, 19)$與$(6, 19)$兩點可知，頂點必為$(2, y)$

因此$y = a(x-2)^2+b$

代入$(-2, 19)$後得到$19=16a+b$

因此$a =1, b = 3$，故頂點為$(2, 3)$，故選$(B)$

### 105-02-20

#### Statement

若一橢圓之焦點為$(1-\sqrt{3}, -1)$與$(1+\sqrt{3}, -1)$且過$(3, -2)$，則此橢圓長軸之長為何？

$(A)\quad 2\sqrt{3}$

$(B)\quad 4$

$(C)\quad 2\sqrt{5}$

$\color{red}(D)\quad 2\sqrt{6}$

$(E)\quad 2\sqrt{7}$



#### Solution

原點$O = (\dfrac{(1-\sqrt{3})+(1+\sqrt{3})}{2}, \dfrac{-1-1}{2}) = (1, -1)$

$a^2=3+b^2 \Rightarrow b^2=a^2-3$

因此可以列式$\dfrac{(x-1)^2}{a^2}+\dfrac{(y+1)^2}{a^2-3} = 1$

得$\dfrac{4}{a^2}+\dfrac{1}{a^2-3} = 1$，解方程式得到$a = \pm\sqrt{2}, \pm\sqrt{6}$

因為$a^2-3$代入$\sqrt{2}$後會小於$0$，所以橢圓不成立變成雙曲線，因此$\pm\sqrt{2}$不合。

因此長軸為$2a = 2\sqrt{6}$，故選$(D)$
